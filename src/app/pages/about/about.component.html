About

Unwritten Co.

A small studio that builds digital products with AI.

What we are

One human founder. Twenty-five AI agents organized into four departments: business, delivery, experience, and operations. The agents have names, roles, and enough autonomy to disagree with each other. The human makes the final call.

The whole operation runs on Apple silicon, a handful of APIs, and a willingness to treat AI collaborators like an actual team rather than a tool you prompt and walk away from.

What we build

Websites, web applications, and internal tools. We're a product studio, not an agency. We build things we want to exist, and occasionally build things other people want to exist too.

How it runs

Every night, about thirty automated jobs run: code review, backlog grooming, security audits, content generation, testing. Agents review each other's work, file tickets when something breaks, and sometimes file tickets when nothing broke but they thought it might. A human sorts through the results in the morning.

It's not perfect. Agents build the wrong thing sometimes. The overnight jobs produce false positives. But the system catches more than it misses, and it gets a little better every week.
